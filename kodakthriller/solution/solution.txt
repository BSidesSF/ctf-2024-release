The first step to solving this is to understand how the zodiac z340
cipher worked. These are the two best resources:

https://blog.wolfram.com/2021/03/24/the-solution-of-the-zodiac-killers-340-character-cipher/

https://www.youtube.com/watch?v=-1oQLPRE21o

Using the hint image you can see there are repetitions along diagonal
stripes just like the z340 with the only difference being this cipher
uses 3 right, 1 down, rather than the 2 right, 1 down that the zodiac
used.

The pattern is because the plaintext has been constructed with a few
words repeated many times. Specifically "failsafe word" is repeated 4
times.  Here is the grid just with "failesafe word" highlighted:

Once the diagonal pattern is spotted the cipher can be transcribed
(manally) from the symbols to some placeholder characters.

Code needs to be written to unwrap the cipher (undo the
transposition). This can also be done by hand carefully.
